# 🏦 Bank Management System

A console + web-based Bank Management System built using Python, Flask, OOP concepts and File Handling.

## 🚀 Live Demo
https://bank-management-system-1-6fan.onrender.com

## 📌 Features
- Create Account
- Login Authentication
- Deposit Money
- Withdraw Money
- Check Balance
- Data stored using JSON file

## 🛠 Technologies Used
- Python
- Flask
- OOP (Object Oriented Programming)
- File Handling (JSON)
- Gunicorn
- Render (Cloud Deployment)

## 📂 Project Structure
- app.py → Flask routes
- bank.py → Business logic
- templates/ → HTML files
- data.json → Database file

## 🎯 Deployment
This project is deployed on Render cloud platform.
